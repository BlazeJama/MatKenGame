# Military Vehicle Recognition Game — Project Summary

## What We're Building
A browser-based quiz game that teaches players to recognize military vehicles.

## Platform Decision: PWA (Progressive Web App)
We decided to build this as a **Progressive Web App (PWA)** because it:
- Works on **iOS, Android, and desktop** from a single codebase
- Can be **installed to the home screen** like a real app — no app store needed
- Works **offline** after the first visit
- Can be **embedded in Google Sites**
- Is **free to host** on GitHub Pages

## Tech Stack
- **Frontend:** React or Vanilla JS + Tailwind CSS (via CDN)
- **Hosting:** GitHub Pages (free, HTTPS)
- **Embedding:** Google Sites iframe
- **Images:** Hosted on GitHub / jsDelivr CDN
